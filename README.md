# Velvet Hotel Website

Single-file, simple-hosting compatible hotel booking website for **Velvet Hotel, Sitamau (Madhya Pradesh, India)**.

## Files
- `velvet.html` – main responsive website + booking + admin panel
- `sitemap.xml` – SEO sitemap

## Features
- Luxury black + gold modern design
- Sticky navigation, hero section, rooms, amenities, gallery, nearby places, reviews, and map
- Booking form with Check-in/Check-out and WhatsApp booking integration
- Floating WhatsApp button
- Local SEO meta tags + LocalBusiness/Hotel schema
- Lightweight admin panel (browser localStorage based):
  - update room prices
  - add room photos/rooms
  - update WhatsApp contact number

## Hosting
Upload `velvet.html` and `sitemap.xml` to any static hosting.
